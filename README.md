Before start run the program please make sure installated JavaFX on NetBeans.

**JavaFX Installation Guide on NetBeans**

**Step 1:** 
Open your NetBeans and click New Project.

**Step 2:**
Select the categories, Java with Maven and the projects, Simple JavaFX Maven ArcheType.
Then, click the Next > button.

**Step 3:**
Tick the option, JavaFX Implementation for Windows (amd64), and click the button, Download and Activate...

**Step 4:**
Click the next > button.

**Step 5:**
Tick the box to agree to the license agreements and click the install button.

**Step 6:**
After installation is completed, click the Finish button.

**Step 7:**
Change the value of javafx-version to 22 and click the finish button. You may rename the project name if you wished.

**Step 8:**
Check your Project and look for the directory/folder named Project Files and open the file, pom.xml.

**Step 9:**
Modify the red boxes version to 22.
